# 🍽️ Food Ordering Dashboard (Excel)

## Overview

This project is an interactive Excel dashboard built to analyze customer food ordering behavior.

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers

## Screenshot

![Dashboard](https://github.com/SujithVarma-ai/Food_Ordering_Dashboard/blob/main/Screenshot%202026-03-29%20220256.png)

## Key Insights

* Mumbai has the highest order volume
* Evening is the peak ordering time
* South Indian cuisine is most preferred

## Files

* Food_Ordering_Dashboard.xlsx → Main dashboard

## Features

* Interactive filtering using slicers
* KPI metrics (Total Orders, Avg Order Value)
* Visual insights using charts
